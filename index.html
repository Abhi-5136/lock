📋 How to Use

To Encrypt Files:

  1. Click "Encrypt" mode
  2. Select files or folder to encrypt (supports any file type)
  3. Enter a strong password (12+ chars, uppercase, lowercase, number, special character)
  4. Click "Encrypt Files"
  5. Download the encrypted files

To Decrypt Files:

  1. Click "Decrypt" mode
  2. Select encrypted files (.enc or .secure)
  3. Enter the password used for encryption
  4. Click "Decrypt Files"
  5. View or download the decrypted files

Note: All encryption happens in your browser. Your files never leave your device.
